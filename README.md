#Sistema de Gestão de Bar

Este é um projeto em Python desenvolvido para gerenciar a entrada de clientes e o consumo de bebidas em um bar. O sistema foca em segurança de dados e experiência do usuário via terminal.

Funcionalidades
* **Verificação de Idade:** Validação automática de maioridade.
* **Tratamento de Exceções:** Sistema protegido contra erros de entrada (ex: impede o fechamento do programa caso o usuário digite letras em campos numéricos).
* **Gestão de Comandas:** Abertura, lançamento de consumo e fechamento de conta com limpeza de cache.

## 🛠️ Tecnologias e Conceitos Aplicados
* **Linguagem:** Python 
* **Estrutura de Dados:** Dicionários (`dict`) para otimização de busca.
* **Lógica de Programação:** Blocos `try/except` para robustez do software.
